# ⚡ ByteForge QR Engine

A futuristic QR Code Generator & Scanner built with Flask and modern cyberpunk UI.

## Features
- Generate QR codes instantly
- Scan QR from image uploads
- Glassmorphism UI
- Purple neon cyber theme

## Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- Pillow, qrcode, pyzbar

## Run Locally
```bash
pip install -r requirements.txt
python app.py
